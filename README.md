1. Password Generator with Strength Checker

 :A simple and secure Python project that generates random passwords and checks their strength. This tool helps users create strong passwords while understanding what makes a password secure.

 :Features

 :Generates fully random passwords

 :Adjustable password length

 :Strength checker (Weak / Moderate / Strong)

 :Uses secure random generation (secrets module)

 :Beginner-friendly, clean, humanised code

 :Runs directly in the terminal

2. How It Works

 :The user enters the desired password length.

 :The program randomly selects characters (letters, digits, symbols).

 :The strength checker evaluates the password based on:

 :Presence of uppercase letters

 :Presence of lowercase letters

 :Inclusion of numbers

 :Inclusion of special characters

3. Overall length

 :The program prints the generated password and its strength rating.

 :Technologies Used

4. Python 3

 :string module

 :secrets (for secure random generation)
